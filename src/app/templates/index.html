Sobre o Projeto

Este projeto foi desenvolvido como trabalho prático para a disciplina de Teoria da Computação. O código do projeto está disponível no Github. Contribuições são bem-vindas.

Máquina Norma

A Máquina Universal Norma possui como memória um conjunto infinito de registradores naturais e três instruções sobre cada registrador: adição e subtração do valor um, e teste se o valor armazenado é zero.
